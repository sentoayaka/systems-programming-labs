## 中文

这个目录保存我在 CS:APP / CMU 15-213 学习中实现的核心源码。公开版本只保留我写的关键 `.c/.h` 文件，不包含课程 handout、driver、trace、PDF、二进制程序和编译产物。

- `malloc/mm.c`：动态内存分配器。
- `cache/csim.c`：cache simulator。
- `cache/trans.c`：矩阵转置 cache 优化。
- `shell/tsh.c`：tiny shell 与 job control。
- `proxy/proxy.c`、`proxy/cache.c`、`proxy/cache.h`：HTTP proxy 与缓存模块。

## English

This directory contains the core source files I implemented while studying CS:APP / CMU 15-213. The public version keeps only my key `.c/.h` files and excludes course handouts, drivers, traces, PDFs, binaries, and build artifacts.

- `malloc/mm.c`: dynamic memory allocator.
- `cache/csim.c`: cache simulator.
- `cache/trans.c`: cache-aware matrix transpose.
- `shell/tsh.c`: tiny shell and job control.
- `proxy/proxy.c`, `proxy/cache.c`, `proxy/cache.h`: HTTP proxy and cache module.
